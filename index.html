  • Radar

    The Radar tool is a passive analysis of twitter, reddit, & youtube users. It can answer the following questions about someone:

    • Does this person interact with toxic people?
    • How toxic is this person?
    • Has this person changed over time?

  • Monitor

    Monitor is a passive toxicity detection system for group discussion. It can be used as an alert system or simply as a histographic tool for these types of situations:

    • Someone is being extra toxic.
    • The discussion tempature is rising.
    • Someone is continually being a toxic community member.

  • API

    For the programmers this provides a stable set of endpoints for programs to query for toxicity information. You can:

    • Determine the toxicity of a piece of text.
    • Store information about a particular user.
    • Query information about the history of someone.
